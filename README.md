# mix_video
把芙芙的角色pv、剧情视频拼成轻链剧情pv

## cut.py
将视频按帧分割成图片，保存到图片集avater中

## generate.py
根据目标视频的每一帧生成对应混合图片

## generate_video.py
由生成的混合图片集生成视频。但是设备性能不行，或者代码没写好，没弄完。
这一步实际是由剪映完成的。

## generate_gpu.py
尝试用gpu加速计算没有完成

## video_add_audio.py
为视频添加音频文件
